# AI Usage Declaration

This document declares how AI tools were used during the development of the Astrologer CRM assignment project.

## Tools Used

- **Cursor IDE with AI Agent** — Primary development assistant for scaffolding, code generation, and documentation

## How AI Was Used

| Area                    | AI Contribution                                              | Human Contribution                          |
| ----------------------- | ------------------------------------------------------------ | ------------------------------------------- |
| Project scaffolding     | Generated Next.js project structure, config files, Prisma schema | Reviewed and approved tech stack choices    |
| Database design         | Proposed Client and Consultation models with relationships   | Validated fields match astrology CRM needs  |
| API routes              | Generated CRUD endpoints with error handling                 | Reviewed API design and request/response shape |
| UI components           | Built reusable components (Button, Badge, Form fields, etc.)  | Reviewed layout and UX flow                  |
| Pages                   | Implemented dashboard, client, and consultation pages        | Verified feature completeness               |
| Seed data               | Created realistic sample Indian client data                  | Approved sample content                     |
| Documentation           | Drafted README, PROJECT_NOTES, and this file                 | Reviewed for accuracy                       |
| Styling                 | Applied Tailwind CSS with astrology-themed purple palette    | Approved visual direction                   |

## What Was NOT AI-Generated

- Final architectural decisions (monolithic Next.js + SQLite)
- Feature scope prioritization for the 48-hour window
- Assignment submission strategy (what to include/exclude)
- Manual testing and verification of all user flows

## Transparency Statement

AI was used as a coding assistant to accelerate development within the assignment timeframe. All generated code was reviewed for correctness, consistency, and alignment with the project requirements. The developer understands the codebase structure, data flow, and can explain and extend any part of the application.

## Percentage Estimate

Approximately **80–90%** of the initial codebase was AI-assisted. The developer provided direction, reviewed output, tested functionality, and made decisions on scope and architecture.
